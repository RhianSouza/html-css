Princípais formatações



Negrito/Destaque

Nesta frase, temos um termo em negrito usando <b> (nao semântico)

Nesta frase, temos um termo em destaque usando <strong> (semântico)



Ítalico/Ênfase

Nesta frase, temos um termo em Ítalico usando <i> (nao semântico)

Nesta frase, temos um termo em ênfase usando <em> (semântico)



Texto marcado

podemos também marcar um texto usando <mark>



Texto grande e texto pequeno

Podemos também criar textos grandes e textos pequenos <big> (big não funciona) usando e <small>



Texto deletado

podemos marcar um texto excluído para indicar que ele deve ser lido mas não considerado usando <del>

Texto inserido

Podemos marcar um texto como inserido para darr ênfase que ele foi adcionado depois usando <ins>



Texto sobrescrito

Para inserir coisas do tipo x20+3 usando <sup>

Texto subscritos

Para inserir coisas do tipo H20 usando <sub>
